Salat (Namaz) timetable for any location around the world!

== Description == 
Displays Salat (Namaz) timetable on your site for any location around the world, based on a variety of calculation methods currently used in muslim communities.

This version is a fork of the original plugin (from version 3.6) that addresses several compatibility issues:

= Key Improvements =
* Fixed PHP 8.0+ compatibility issues
* Resolved styling conflicts with Elementor
* Added new color schemes including burgundy theme
* Improved default options handling
* Enhanced table styling compatibility

= Features =
* Prayer times for any location worldwide
* Multiple calculation methods
* Customizable color schemes
* Hijri date display option
* Multilingual support
* Widget and shortcode implementation
* Full Elementor compatibility
* Responsive design

= Usage =
* Install and activate the plugin
* Navigate to: Settings > Salat Times
* Configure your location and preferences
* Use widget or shortcode to display

= Color Schemes =
* Burgundy (New)
* Black
* Blue
* Light Blue
* Gray
* Green
* Orange

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/salat-times` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Salat Times screen to configure the plugin
4. Add widget or use shortcode [daily_salat_times]

== Frequently Asked Questions ==

= How to use shortcode? =

Use shortcode: [daily_salat_times]

= How to change location? =

Go to Settings > Salat Times. Then set your location (latitude, longitude, timezone).

= How to change calculation method? =

Go to Settings > Salat Times. Then select your preferred calculation method.

= How to change color scheme? =

Go to Settings > Salat Times. Then select your preferred color scheme from the dropdown menu.
